== Description ==

WooCommerce has lots of filters to let you customize button text, labels, and more, for that you have to write code to use them. This plugin provides you setting page where you can customize Product loop Setting, Shop page setting, Product description.

<b>Woo Shop Page Customizer</b> plugin also allows us to change button color, text, and design, all of us can do it without needing to write any code or modify any templates, which is helpful for a quick change.

There mainly four customizer options Product Loop Setting, Shop Page Setting, Design Elements, and Product Description.

In Product Loop Setting you can change Simple Product Add to cart button text, Variable Product Add to cart button text, Grouped Product Add to cart button text, Out of Stock button text. Change Sale Flash badge text. Options to hide product meta like Sale Flash, Add to cart, Price, Sale Price, Variable Price, Rating, Image.

Option to display shop now button in product loop. Option to show Stock quantity. Also, customize stock amount text. You can set custom low stock alert text.

In Shop Page Setting you can shop page layout. You can choose how many products to show on a page. Can also change how many products to show per row. There are other options like breadcrumbs, shop title, shorting dropdown, and shorting results that can also be hidden.

In Design Elements you can customize Add to cart button & Shop now font size, padding, etc. Can change Sale Flash font size. font-weight. Also can customize font color and font size of product title and price.

We do support our free plugins and extensions, but please understand that support for premium products takes priority. We can immediately solve any issue of yours.

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from the Plugins page. Or directly go to Add Plugins page and search "WooShop Page Customizer" and install then active. 

After Plugin Active go to WooCommerce-> Woocommerce Shop Page Customizer. Or go to Plugins-> Installed Plugins-> Shop Page Customizer WooCommerce -> Setting.
